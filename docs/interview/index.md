# Interview Preparation

Senior-level Java Backend Developer interview guide — 63 questions with detailed model answers covering Core Java, Spring Boot, Hibernate, Databases, REST API, Microservices, Kafka, AWS, and more.

## How to use this section

Each question follows the same format:

- **Quick answer** — a concise answer you can give in 1–2 minutes
- **Full model answer** — collapsible deep-dive with all edge cases
- **Golden Tip** — what separates a good answer from a great one
- **Follow-up questions** — what the interviewer asks next

> Self-test technique: read the question, think through your answer, then expand the details block to compare.
